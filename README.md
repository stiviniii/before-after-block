# Before After Image Compare Block

A high-performance, lightweight, and fully customizable **WordPress Gutenberg block** for comparing images with an interactive slider. Built with **Modern Block API v3** and **Pointer Events** for a seamless experience on desktop and mobile.

## 🚀 Key Features

- **High Performance**: Optimized with `requestAnimationFrame` for 60fps interaction and `IntersectionObserver` for lazy initialization.
- **Mobile First**: Native touch support using Pointer Events—no jQuery or heavy libraries required.
- **Accessibility Centric**: ARIA labels, semantic markup, and full keyboard navigation (Arrow keys, Home, End).
- **Customizable Handles**: Use beautiful SVG icons for the slider handle with customizable colors and sizes.
- **Responsive Heights**: Tailor the block height separately for Desktop, Tablet, and Mobile devices.
- **Real-time Preview**: Intuitive editor experience with immediate image previews in the placeholder.
- **Developer Friendly**: Built with `@wordpress/scripts`, ESNext, and CSS Variables for easy styling.

## 🛠️ Installation

1. Upload the `before-after-block` folder to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. In the Gutenberg editor, search for **"Before After Image Compare"** and add it to your page.

## 🎨 Customization Options

- **Image Settings**: Toggle between `Cover` and `Contain` fit modes.
- **Label Settings**: Customize "Before" and "After" text, background colors, text colors, and unique positions for each label.
- **Slider Settings**: Set the default starting position, enable smooth animations, and toggle arrow visibility.
- **Handle & Divider**: Adjust the thickness, color, and size of the slider handle and the divider line.
- **Block Styling**: Add optional shadows and border-radius to the entire container.
